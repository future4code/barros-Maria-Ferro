Projeto Lista JavaScript

O projeto Lista de Exercícios - Introdução ao JavaScript foi a primeira lista de exercícios que fizemos durante o curso na Labenu, na matéria de JavaScript. Tivemos acesso aos enunciados como parte do material do curso, e então recebemos um documento com testes automatizados e uma arquivo .js de inserção de código, na qual deveríamos criar a lógica para cada exercício. Para verificar se estava funcionando, acessávamos o arquivo .html, que executava os testes. Nas imagens abaixos podemos ver mais sobre a página e o resultados dos testes, bem como um pouco da execução dos exercícios no VSCode.
