Projeto BlackJack

O Projeto BlackJack foi realizado como parte do curso da Labenu e sob orientação deles. Fez parte de um exercício de fixação da matéria de JavaScript, em que o código de sorteio de cartas e demais informações foram fornecidos para nós e precisávamos criar a lógica necessária para que o jogo rodasse como deveria e imprimisse as informações no console. Nas imagens abaixo, podemos ver que a primeira interação com o usuário é perguntar se ele gostaria de iniciar uma rodada. Caso diga que sim, o resultado já é impresso no console. Caso diga que não, a mensagem que aparece no console é de que o jogo acabou.
